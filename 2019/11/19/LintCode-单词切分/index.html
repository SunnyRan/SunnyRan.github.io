LintCode- 单词切分

单词切分

给出一个字符串s和一个词典，判断字符串s是否可以被空格切分成一个或多个出现在字典中的单词。

样例
s = “lintcode”

dict = [“lint”,”code”]

返回 true 因为”lintcode”可以被空格切分成”lint code”
